# 木漏れ日 · Komorebi

**Komorebi** — the interplay of light filtering through leaves.

A Japanese-inspired pastel component library. Built on React, Vite, TypeScript, and shadcn/ui with a distinctive Japanese wabi-sabi aesthetic.

## Quick Start

```tsx
import { Button } from '@stiaan/komorebi'

export function App() {
  return (
    <Button variant="sakura">
      Blossom
    </Button>
  )
}
```

Import the stylesheet once at your app entry:

```tsx
import '@stiaan/komorebi/styles'
```

## Design Principles

**Wabi-sabi** — embrace subtle imperfection. Rounded but not bubbly. Pastel but not washed out.

**Distilled** — every component does one thing well. No hidden complexity.

**Accessible** — built on Radix UI primitives. Keyboard navigable, ARIA-compliant, focus-visible by default.

**TypeScript-first** — strict mode, `noUncheckedIndexedAccess`, exported prop interfaces, CVA variant types.
